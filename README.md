# MovieDatabase

###Instalation:
1. `make start` - it will build docker image and install all requirments and will start app.
2. `make migrate` - to run migrations
3. App is running on localhost:8000



####More make commands:
1. `make test` - to run tests
2. `make makemigrations` - to run makemigrations
3. `make stop` - to stop the container
